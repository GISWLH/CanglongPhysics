# 超级干旱预测

超级干旱预测产品采用多尺度综合指数（CMI）来综合表征不同时间尺度(不同类型)干旱的整体叠加程度和极端性，这是识别和预警超级干旱的核心指标。同时，产品还提供3个月、6个月和12个月这三个时间尺度的标准化降水蒸散指数（SPEI），用于刻画各个单独时间尺度上的干旱强度，辅助分析干旱在不同尺度上的演变特征。下图显示了模式2026年2月起报预报2026年4月的CMI以及3、6、12个月时间尺度的SPEI分布。  
预测结果显示，2026年4月，我国江西东部、福建西北部等地可能面临一定的干旱，吉林西部等地也需对超级干旱的可能发生进行防范。  
而综合其他时刻的各干旱指数预测结果（详见https://superdrought.com/prediction.html）来看，在2026年3月，我国东南地区超级干旱风险最高，多尺度干旱叠加效应显著，需提前做好防旱准备；而在2026年3-5月，我华北-河套地区以短中期干旱为主，3月气象干旱发展，4-5月农业干旱持续，需重点关注对春耕生产的影响。由于水文系统整体偏湿，通过水利工程调度和农田灌溉或可有效缓解旱情。
超级干旱预测结果同样发布于超级干旱在线平台预报产品试运行上线。  
### ————关于超级干旱————
超级干旱，是在多个时间尺度上同时发生极端干旱的现象。其在物理意义上表征水文水资源各部分的复合水亏缺。超级干旱综合多尺度指数（CMI）用于描述多尺度干旱的整体极端程度、罕见程度并识别超级干旱。参考文献：Wang Lin, Huang Gang, Chen Wen, Wang Ting. (2023). Super drought under global warming: concept, monitoring index and validation. Bulletin of the American Meteorological Society, 104(5), E943—E969.王林, 陈文, 黄刚, 等. (2024). 中国西南超级干旱的变化特征和多尺度异常叠加效应分析. 中国科学地球科学, 54(7), 2114—2132.
